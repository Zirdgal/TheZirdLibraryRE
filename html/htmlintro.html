HTML Introduction!

A basic introduction to HTML!

What is HTML?

HTML or Hypertext Markup Langauge, is the standard markup langauge for documents designed to be displayed in a web browser. It is often assisted by technologies such as Cascading Style Sheets (CSS) and sripting langagues such as JavaScript or in this websites case, also Lua.


How do I start writing in HTML?

Very easy! Make any file, anywhere on your computer and add the .html tag at the end and thats it! You have succsesfully created a HTML file! But of course there is nothing in there. So use your text editor of choice if you don't have any idea what that means, your windows notepad will do just fine! Open the file in chosen text editor and now we can get to the interesting stuff!


The Basics!

Now then, you have opened the HTML in your desired text editor, now we can start coding! First of any html file you need to tell your file its HTML version, currently we are on HTML 5 which would be <!DOCTYPE html>. If you want to write this for yourself, you must have the brackets aswell, because the text inbetween those brackets are what we call a tag or a block depending on what background your from. Next we need the <html> tags! Now here comes a nuance that you will see quite a lot throughout the langauge. The tag <html> has a closing tag which looks like this: </html>. The dash is what makes the tag a closing tag, it is required to define the contents of <html> as you will see later!
